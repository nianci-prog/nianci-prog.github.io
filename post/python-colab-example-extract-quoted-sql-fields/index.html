Python / Colab Example: Extract Quoted SQL Fields with Line Breaks

Oct 1, 2025

Overview

This note demonstrates how to use Python in Google Colab to extract text enclosed in double quotes from SQL SELECT statements in a CSV file.
The extracted values are placed in a new column with each value on a separate line, ready for Excel display.

1. Import Packages

import pandas as pd
import re
from google.colab import files
import csv  # For CSV quoting

2. Upload CSV

uploaded = files.upload()  # Upload your CSV file
file_name = list(uploaded.keys())[0]  # Get uploaded file name

Note: The CSV must contain at least one column with SQL queries, e.g., query.

3. Read CSV

df = pd.read_csv(file_name)

4. Extract Quoted Text and Add Line Breaks

def extract_quotes(text):
    if pd.isna(text):
        return ""
    quotes = re.findall(r'"(.*?)"', text, re.DOTALL)  # Extract text inside double quotes
    return "\n".join(quotes)  # Add line breaks

	•	re.DOTALL allows regex to match across multiple lines.
	•	"\n".join(quotes) ensures each quoted value is on a new line.

Apply to your column:

df['Quoted_Fields'] = df['query'].apply(extract_quotes)

A new column Quoted_Fields is created containing each quoted text on a separate line.

5. Save as New CSV

output_file = "with_quotes.csv"
df.to_csv(output_file, index=False, encoding="utf-8-sig", quoting=csv.QUOTE_ALL)
  • quoting=csv.QUOTE_ALL ensures Excel properly displays line breaks inside quoted cells.

6. Download Result

files.download(output_file)

7. Completion Message

print("Done! 'with_quotes.csv' has been generated with each quoted text on a new line.")

Key Takeaways

  • Use re.findall(r’"(.*?)"’, text, re.DOTALL) to extract text inside double quotes.
  • Use “\n”.join(quotes) to add line breaks.
  • csv.QUOTE_ALL ensures Excel displays the newlines correctly.
  • Colab allows direct CSV upload, processing, and downloading.
